# Cloud File Storage System
# Cloud-Based File Storage System

A secure cloud-based file storage backend application developed using Node.js, Express.js, and MongoDB Atlas. This project allows users to upload, manage, and download files securely with authentication support.

---

## Features

- User Authentication using JWT
- Secure File Upload System
- File Download Functionality
- MongoDB Atlas Database Integration
- REST API Architecture
- Middleware-based Authorization
- File Metadata Management
- Organized Backend Structure

---

## Tech Stack

- Node.js
- Express.js
- MongoDB Atlas
- JWT Authentication
- Multer
- REST APIs
